Installs Git for Windows using the Windows Package Manager.

## What it does

Uses `winget` to install the latest stable release of Git for Windows silently, accepting all source agreements automatically.

## Requirements

- Windows 11
- winget (App Installer) available
- PowerShell 5.1 or later
